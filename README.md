# PickFilm

Web application for choosing a movie to watch
## Links

- Project [PickFilm](https://pick-film.vercel.app/)
- Repository [Repository](https://github.com/XJloTT4eH9l/PickFilm)

## Tech stack
- React
- TypeScript
- React router dom
- Redux tool kit
- Sass(SCSS)
- REST API (used [TMBD API](https://www.themoviedb.org/))
